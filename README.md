# ospov0
An OSPO website template by v0.dev https://ospov0.vercel.app/

## Guide

This project was initiated by v0.dev with Next.js.

Install dependencies 
```shellscript
npm install
```

start for dev 
```shellscript
npm run dev
```

build 
```shellscript
npm run build
npm run start
```
